# Third Project Resilia
 
### Third team project in resilia course.


### Dependencies

```sh
- React
- Axios
- MUI CORE
```

### Installation

Clone this repo and install the dependencies

```sh
$ git clone https://github.com/squad4resilia/third-project-squad
$ cd third-project-squad/
$ npm install
$ npm run dev
```

### Methods

```sh
- GET /produtos
- POST /produtos
- PATCH /produtos
- DELETE /produtos
```
